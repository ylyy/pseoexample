React 入门指南

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 开发并维护。它采用组件化开发模式，让构建复杂的 UI 变得简单高效。

什么是 React？

React 是一个声明式、高效且灵活的 JavaScript 库，用于构建用户界面。它让你可以通过组合小的、可复用的组件来构建复杂的应用。

核心概念

1. 组件 (Components)

组件是 React 应用的基本构建块。每个组件都是一个独立的、可复用的代码单元。

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

2. JSX

JSX 是 JavaScript 的语法扩展，让你可以在 JavaScript 中编写类似 HTML 的代码。

const element = <h1>Hello, World!</h1>;

3. Props

Props（属性）是组件之间传递数据的方式，类似于函数的参数。

4. State

State 是组件内部的数据，当 state 改变时，组件会重新渲染。

开始使用

最简单的开始方式是使用 Create React App：

npx create-react-app my-app
cd my-app
npm start
