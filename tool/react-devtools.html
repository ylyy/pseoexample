React DevTools

React DevTools 是 Facebook 开发的官方浏览器扩展，用于调试 React 应用。它提供了强大的组件检查、性能分析和状态调试功能。

安装

React DevTools 支持 Chrome、Firefox 和 Edge 浏览器：

  • Chrome: 在 Chrome Web Store 搜索 "React Developer Tools"
  • Firefox: 在 Firefox Add-ons 搜索 "React Developer Tools"
  • Edge: 在 Edge Add-ons 搜索 "React Developer Tools"

主要功能

1. 组件树查看

查看应用的组件层次结构，检查组件的 props 和 state。

2. Props 和 State 编辑

实时编辑组件的 props 和 state，快速测试不同状态下的 UI 表现。

3. 性能分析

使用 Profiler 标签页分析组件渲染性能，找出性能瓶颈。

4. 组件搜索

快速搜索和定位特定组件。

使用技巧

  • 使用 $r 在控制台访问选中的组件实例
  • 启用 "Highlight updates" 查看组件重渲染
  • 使用 Profiler 记录性能数据
