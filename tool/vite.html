Vite - 下一代前端构建工具

Vite 是一个快速的前端构建工具，提供极速的开发体验和优化的生产构建。

快速开始

npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev

主要特性

  • 极速的开发服务器启动
  • 热模块替换 (HMR)
  • 优化的生产构建
  • 原生 ES 模块支持
