Create React App

Create React App 是官方推荐的创建单页 React 应用的工具，无需配置即可开始开发。

快速开始

npx create-react-app my-app
cd my-app
npm start

主要特性

  • 零配置，开箱即用
  • 包含开发服务器
  • 自动优化生产构建
  • 支持 TypeScript

项目结构

my-app/
  ├── public/
  ├── src/
  │   ├── App.js
  │   ├── index.js
  │   └── ...
  └── package.json
