React Server Components 探索 LAB

React Server Components (RSC) 是 React 团队推出的新特性，允许在服务器上渲染组件，减少客户端 JavaScript 包大小。

核心概念

Server Components 在服务器上运行，可以直接访问数据库和文件系统，无需发送到客户端。
