# 🚀 快速开始 - 3 步部署到 GitLab

## 第一步：创建 GitLab 项目 ⏱️ 2分钟

1. 访问 https://gitlab.com 并登录
2. 点击右上角 **"+"** → **"New project"**
3. 项目名填：`pseoexample`
4. 点击 **"Create project"**
5. **复制仓库地址**（类似：`https://gitlab.com/你的用户名/pseoexample.git`）

---

## 第二步：推送代码 ⏱️ 1分钟

在终端（Terminal）运行以下命令：

```bash
# 1. 进入项目目录
cd /Users/test/pseoexample

# 2. 添加远程仓库（替换成你的实际地址）
git remote add origin https://gitlab.com/你的用户名/pseoexample.git

# 3. 推送到 GitLab
git push -u origin master
```

**如果提示需要密码：**
- 用户名：你的 GitLab 用户名
- 密码：你的 GitLab 密码（或 Personal Access Token）

---

## 第三步：等待部署 ⏱️ 3-5分钟

1. 在 GitLab 项目页面，点击 **"CI/CD"** → **"Pipelines"**
2. 等待任务变成 **绿色 ✅**（表示成功）
3. 点击 **"Settings"** → **"Pages"**
4. 你会看到网站地址：`https://你的用户名.gitlab.io/pseoexample/`
5. 打开这个地址，网站就可以访问了！

---

## ✅ 完成！

就这么简单！你的网站已经部署成功了。

**下次更新内容：**
```bash
git add .
git commit -m "更新内容"
git push
```
GitLab 会自动重新部署，几分钟后就能看到更新。
